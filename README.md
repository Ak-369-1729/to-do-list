## How to Use

1. **Add a Task** - Type your task in the input field and click "Add" or press Enter
2. **Complete a Task** - Click the checkbox next to a task to mark it as complete
3. **Filter Tasks** - Use the filter buttons to view All, Active, or Completed tasks
4. **Delete a Task** - Click the delete button (X) on any task to remove it
5. **Clear Completed** - Click "Clear Completed" to remove all finished tasks at once
